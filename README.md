# SkillSwap

**SkillSwap** is a responsive web platform that allows users to exchange skills with each other without any monetary payment.  
Each user can teach a skill they already know in exchange for learning a different skill from someone else.

Built with **Next.js** and **Tailwind CSS**, SkillSwap follows a **mobile-first** approach and is fully responsive across all screen sizes.

---

## ✨ Project Overview

SkillSwap was designed to create a simple and modern environment where people can connect, share knowledge, and grow together through skill exchange.

The platform focuses on:
- Easy skill discovery
- Smart matching between users
- Clean and responsive user experience
- A modern UI inspired by soft lavender tones, indigo accents, and subtle gradient styling

---

## 🚀 Features

### Completed Pages
- **Landing Page**  
  A modern homepage that introduces the platform and highlights its value.

- **Match Page**  
  A page designed to help users discover and connect with suitable skill partners.

- **Skills Page**  
  Displays available skills and helps users browse what they can teach or learn.

### Planned Pages

- **Chat Page**

## 🛠️ Tech Stack

### Current Technologies Used
- **Next.js** – React framework for building fast and scalable web apps
- **Tailwind CSS** – Utility-first CSS framework for rapid and responsive styling
- **React** – UI component-based development
- **JavaScript / TypeScript** – For building app logic and maintainable code

### Recommended Technologies to Continue the Project
These technologies can help complete and scale SkillSwap efficiently:

- **NextAuth.js** – Authentication for login/register
- **Firebase / Supabase** – Backend services, auth, and database
- **Prisma** – ORM for structured database management
- **PostgreSQL / MongoDB** – Database options depending on your data structure
- **Socket.io** – Real-time chat and instant communication
- **React Hook Form** – Form handling for register/login pages
- **Zod** – Form validation and schema checking
- **TanStack Query** – Efficient server state management
- **Zustand / Redux Toolkit** – Global state management
- **Framer Motion** – Smooth animations and transitions
- **Lucide React** – Clean and lightweight icons
